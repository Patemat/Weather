# Weather
Website
# Веб-сайт прогноза погоды

Проект представляет собой веб-сайт, который предоставляет актуальную информацию о погоде в различных городах мира. Сайт использует API для получения данных о текущей погоде, температуре, влажности, скорости ветра и других параметрах.

## Структура проекта

- `index.html` - Главная страница веб-сайта.
- `stile/` - Стили CSS для оформления страниц.
- `main/` - JavaScript файлы для обработки запросов и отображения данных.
- `assets/` - Изображения и другие ресурсы для сайта.
- `README.md` - Документация по проекту.

## Установка 

1. Клонируйте репозиторий:
    ```bash
    git clone https://github.com/Patemat/Weather
    ```

2. Перейдите в папку проекта:
    ```bash
    cd Weather-website
    ```

3. Откройте файл `index.html` в браузере.

    Для локальной разработки можно использовать сервер, например, с помощью Python:
    ```bash
    python -m http.server
    ```

    После этого откройте браузер и перейдите по адресу: `http://localhost:8000`

## Использование

1. Введите название города в поле поиска на главной странице.
2. Нажмите кнопку "Получить прогноз", чтобы увидеть информацию о текущей погоде для выбранного города.

### Пример использования:

- Введите город, например: `Moscow`, `New York`, `Paris`.
- Сайт покажет температуру, влажность, скорость ветра и другие данные.

## Технологии

- HTML
- CSS
- JavaScript (для обработки запросов и динамического обновления страницы)
- [OpenWeatherMap API](https://openweathermap.org/api) (для получения данных о погоде)

## Тестирование

Для тестирования функциональности сайта рекомендуется проверить работу поиска и отображения данных для разных городов.

1. Откройте сайт в браузере.
2. Введите несколько популярных городов и проверьте, что данные отображаются корректно.
